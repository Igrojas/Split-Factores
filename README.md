# Split Factores

Proyecto de simulación de Split Factor con análisis Monte Carlo para circuitos de flotación.

## Descripción

Este proyecto contiene simulaciones de circuitos de flotación utilizando el método de Split Factor con análisis de Monte Carlo para evaluar la variabilidad de los resultados.

## Estructura del Proyecto

- `Sims caso base/`: Scripts de simulación del caso base
- `Split_factor_MC_caso_*.py`: Scripts de simulación para diferentes casos
- `test_dilucion.py`: Script de pruebas de dilución

## Requisitos

- Python 3.10+
- pandas
- openpyxl
- numpy
